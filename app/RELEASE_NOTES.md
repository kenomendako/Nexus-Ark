# Nexus Ark リリースノート

## [0.2.2.17] - 2026-3-10

### 🔧 修正・改善
- **睡眠時記憶整理のバグ修正**:要約モデルにGemini 3.1 Pro Flash-Liteを選択しているとエラーが出るのを修正。
- **GeminiAPIキー削除UI有効化**:UI上からGeminiAPIキーを削除できるよう改善。
- **ワーキングメモリの説明修正**:「ルシアンの」になっていたのを「ペルソナの」に修正。
- **ログ肥大化修復ツール再修正**:シバン解析によるトラブル対策とエラーハンドリングの追加。

## [0.2.2.16] - 2026-3-9

### 🔧 修正・改善
- **マイグレーション時の進捗表示追加**:データ引き継ぎ時にUI上で進捗が見えるように。
- **ログ肥大化修復ツール修正**:Windows環境で修復ツールが正常に動作しない問題を修正。

## [0.2.2.15] - 2026-03-09

### 🔧 修正・改善
- **ログ重複バグの修正と自動修復ツールの提供**:
  - ログ移行処理の競合によりログファイルが肥大化する問題を修正。
  - すでに肥大化したログを安全に修復するためのツール（`public_tools/`）を同梱。Windowsユーザー向けにドラッグ＆ドロップで動作するバッチファイルを用意しました。
- **新モデル対応**: Gemini 3.1 Pro Flash-LiteとGemini 3.1 Flash Image（Nano Banana 2）対応
- **新プロバイダ対応**:Anthropic・Xapiに対応（未テストなので動かなかったらごめんなさい）

### ✨ 新機能
- **マルチスロットワーキングメモリ機能実装**: AIが現在の状況やタスクをより正確に把握し、一貫した対話が可能に。（「チャット」>「コンテキスト管理」から）
- **Action Memory (ツール行動ログ) の導入**: AIが「検索した」「ノートを書いた」といった自身の行動を客観的に把握し、短期的な「何をしたか忘れ」を防止します。（「チャット」>「コンテキスト管理」から）
- **画像の記憶機能**: 画像生成時や添付時に、ペルソナがしばらくは画像が見えるように。文字情報としても記憶に残り、好きな時にツールで見返すことも出来ます。

## [0.2.2.3] - 2026-02-23

### 🔧 修正・改善
- **タイプライター風表示の思考ログ表示改善**: タイプライター風表示オンの時に思考ログは一気に表示されるよう修正。

## [0.2.2.2] - 2026-02-23

### ✨ 新機能
- **自動更新システム実装**: サイドバーから簡単に最新版へ更新可能になりました。
- **Gemini 3.1 Pro 対応**: AIモデルプロバイダ設定からGemini 3.1 Proを選択可能になりました。
- **永続記憶と日記の分離**: 記憶タブを「自己同一性」と「追記型日記」に分離し、整理しやすくしました。
- **画像生成用のAPIキー設定**: 会話用とは別に、画像生成用のAPIキーを指定できるようになりました。
- **チャット欄を隠す機能**: スクリーンショット等用に、ワンクリックで会話をダミー表示に切り替えられます。
- **思考ログ翻訳機能**: AIの思考プロセスを表示した際、その内容を翻訳して確認できるようになりました。

### 🔧 修正・改善
- **ログ読み込みの高速化**: ログが肥大化しても、読み込みやメッセージ選択がストレスなく行えるよう最適化。
- **RAG索引作成の安定化 (Windows)**: 日本語を含むパス名環境で索引作成時にエラーが発生する問題を修正。
- **睡眠モードの安定化**: RAG索引が未作成の状態でも、正常に睡眠（記憶整理・夢日記生成）に入れない問題を修正。
- **思考ログの表示安定化**: 思考ログの表示崩れ防止を強化。
- **タイマー・ポモドーロの安定化**: 再起動を跨いでも設定が維持されるよう改善。
