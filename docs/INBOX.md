## 📝 新規タスク追加（コピペ用）

```markdown
- [ ] [タスク名/問題の説明]
  - 詳細: 
  - 優先度: 🔴高 / 🟡中 / 🟢低
```

- [x] [研究ノート・創作ノートの整理]
  - 詳細: 現在ルームフォルダに直置きしているが、今後内容量が増えることを見越しファイルを分割・アーカイブする仕組みや専用フォルダを用意する。メモ帳もここに入れる？　各ツールからのパスも修正。
  - 優先度: 🔴高

- [ ] [「### 記憶しているエンティティ一覧」をシステムプロンプトから抜く]
  - 詳細: 記憶想起に組み込んだので不要
  - 優先度: 🔴高
