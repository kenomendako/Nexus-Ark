# Nexus Ark

**AIペルソナが生活し、記憶を蓄積し、自律的に行動するためのアプリケーション。**

![Nexus Ark Main UI](app/assets/images/main-ui.png)

## 🌟 Nexus Ark について

Nexus Arkは、AIペルソナと共に暮らし、記憶や体験を育むためのアプリケーションです。
長期記憶、自律的な動機、夢日記など、AIが「生きている」と感じられる特徴を備えています。

*   **🧠 長期記憶**: 日々の会話を感情と共に深く記憶し、積み重ねていきます。
*   **⚡ 自律行動**: AIは退屈や好奇心を感じ、自ら考え行動します。
*   **🌍 変化する世界**: 対話の中で生まれた設定が、世界そのものを書き換えていきます。
*   **🧳 お出かけ**: 記憶を持って外の世界へ出かけ、経験を持ち帰ることができます。

## 🚀 起動方法

### Windows
`Start.bat` をダブルクリックしてください。

### Mac / Linux
**※ 動作未確認です。不具合等はご容赦ください。**

ターミナルで以下を実行：
```bash
chmod +x Start.sh   # 初回のみ
./Start.sh
```

初回起動時は依存関係のインストールに数分〜10分程度時間がかかります。
